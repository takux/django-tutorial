## 本リポジトリについて

- 各セクション終了時点ごとのソースコードを各 sec フォルダへ掲載しています。必要な箇所にご活用ください。

- 仮想環境の venv フォルダも Github には UP していません。使用する際はご自身の環境で仮想環境を構築してください。
